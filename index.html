JavaScript Flashcards

  • JavaScript

    • Programming language of the web. jQuery is built out of JavaScript.

  • Operators

    • Compare two things to determine whether they're equal or not equal.

  • Variable Naming Conventions

    • Lower camel-case, specific, no abbreviations.

  • Functions

    • Isolated code blocks that can later be called upon.

  • Methods

    • Actions that are done on objects.

  • Booleans

    • A binary data type, which can be only True or False.

  • Undefined

    • Something not defined.

  • NaN

    • Abbreviation of not-a-number.

  • Escape

    • '\' character. Used to include special characters in strings, to escape the conventions of a programming language.

  • Alert

    • A function in Javascript to create pop-ups, yay!
